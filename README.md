# NexumPOS | Inteligencia Artificial para tu Negocio 🚀

NexumPOS es una landing page de alto impacto diseñada para presentar soluciones tecnológicas avanzadas para PYMEs en México, con un enfoque especial en lavanderías y comercios multi-giro.

![NexumPOS Preview](assets/img/hero.png)

## ✨ Características Principales

- **Laboratorio de IA Interactivo**: 
  - **Auditoría Nexum**: Detección inteligente de fugas e inconsistencias.
  - **Calculadora de ROI**: Análisis de ahorro de tiempo y retorno de inversión.
  - **Smart Marketing**: Generador de campañas de WhatsApp persuasivas.
- **Solución Multi-Giro**: Gestión de inventarios, proveedores y portal web administrativo.
- **Innovación en Cobro**: Sección dedicada a la **Doble Pantalla** para transparencia absoluta con el cliente.
- **Diseño Premium**: Interfaz moderna con Tailwind CSS, glassmorphism, animaciones fluidas y optimización móvil.
- **Integración con Groq AI**: Utiliza el modelo `llama-3.3-70b-versatile` para respuestas instantáneas.

## 🛠 Tecnologías Utilizadas

- **Frontend**: HTML5, Tailwind CSS, JavaScript (Vanilla ES6+).
- **Iconos**: FontAwesome 6.
- **Tipografía**: Google Fonts (Inter & Outfit).
- **IA Engine**: Groq Cloud API.

## 🚀 Instalación y Uso

1. Clona este repositorio:
   ```bash
   git clone https://github.com/foxsolid23df-IA/WEBNEXUSPOS.git
   ```
2. Abre `index.html` en tu navegador o utiliza un servidor local (Live Server recomendado).

### Configuración de la IA

Para activar las funciones de Inteligencia Artificial, debes configurar tu API Key de Groq:
1. Abre `assets/js/script.js`.
2. Busca la constante `GROQ_API_KEY`.
3. Reemplaza el valor con tu propia llave de [Groq Cloud](https://console.groq.com/).
